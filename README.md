# 🌟 Personal Portfolio Website

## 📌 Overview
This is a modern, responsive personal portfolio website built with HTML and CSS. It showcases my skills, services, and professional background as a Full-Stack Developer. The design is clean, professional, and fully responsive across all devices.

## 🎯 Project Requirements
- ✅ Single HTML file with semantic structure
- ✅ Separate CSS file (no inline styles)
- ✅ Fully responsive (desktop, tablet, mobile)
- ✅ CSS Flexbox & Grid layouts
- ✅ Consistent color scheme (#7cf03d, #4ecdc4, #1f242d)
- ✅ Header, About, Skills, Services, Contact sections

## ✨ Features
- **Modern UI Design** - Clean and professional
- **Typing Animation** - Dynamic role display
- **Responsive Layout** - Works on all devices
- **Social Media Integration** - Quick access links
- **Interactive Elements** - Smooth hover effects
- **Animated Background** - Page loading animation
- **Rotating Profile Border** - Eye-catching effect

## 🛠️ Technologies
- HTML5
- CSS3 (Animations, Flexbox, Grid)
- Font Awesome Icons
- Google Fonts (Poppins)
